## ü¶ñ What is trex?
trex is a template manager in the form of a CLI app. You can create, organize, and clone-from template directories. It doesn‚Äôt matter if the template is a directory on your machine or a GitHub repository: with trex you only need a few seconds to actually get to coding. Not only that, but trex is really easy to use.

## ‚òÅÔ∏è Install trex
**Make sure you have Python 3.9 or above installed on your system.**

trex is available on PyPi and can be installed via pip with the following command:
```

```
Check if the installation was successful:
```
trex version
```
